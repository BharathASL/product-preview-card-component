### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/BharathASL/product-preview-card-component.git
   ```
2. Open the project folder:
   ```bash
   cd product-preview-card-component
   ```
3. Open `index.html` in your preferred browser.
